
## Steps to setup starter template

1.  Clone the starter_Project

````
git clone https://github.com/Palkunta/Express-Typescript-Starter-Project.git <project_name>
````

2. Move into the Folder Structure

```
cd <project_name>
```

3. Install npm dependendies

```
npm i
```

4. Create a new .env file in the root directory and add the 'PORT' env variable

```
echo PORT=3000 >> .env
```

5. Start the express server

```
npm run dev
```
